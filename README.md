# TEKEAT — 오늘 뭐먹지

사내 점심 식당을 빠르게 탐색하기 위한 웹 프로젝트입니다.

## 이번 최신화/고급화 포인트
- 카테고리 + 검색 + 정렬(거리/시간/이름) 조합 탐색
- 폐업 식당 포함 여부 토글
- "🎲 오늘의 추천" 랜덤 픽 기능
- 필터 상태/결과 수를 요약하는 상단 요약 배지
- 식당 리스트 카드형 UI 및 상세/지도/이미지 영역 개선

## 실행 방법
정적 파일 프로젝트이므로 아래처럼 바로 실행할 수 있습니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속 후 사용하세요.
